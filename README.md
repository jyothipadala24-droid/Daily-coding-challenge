# 📊 Sales Data Analysis Dashboard (Excel)

## 📌 Project Overview

This project is part of the **50-Day Data Analyst Coding Challenge – Day 1 (Excel)**.
The goal of this project is to analyze raw sales data and build an **interactive dashboard in Microsoft Excel** to identify key business insights.

The dashboard helps businesses quickly understand:

* Top performing products
* Revenue trends
* Sales performance
* Key insights from the dataset

---

## 📂 Dataset

The dataset used in this project contains **sales transaction information**.

### Columns in Dataset

* Order ID
* Order Date
* Product Name
* Category
* Region
* Quantity
* Revenue
* Customer Information

This dataset was cleaned and organized before creating visualizations.

---

## 🧹 Data Cleaning Process

The following steps were performed to clean the data:

1. Removed duplicate records
2. Handled missing values
3. Corrected formatting issues
4. Converted data into **Excel Table format**
5. Verified numerical columns for calculations

Tools used:

* Excel functions
* Sorting & Filtering
* Table formatting

---

## 📊 Analysis Performed

Key analyses done in the project:

* Identified **Top 5 Products by Revenue**
* Calculated **Total Revenue**
* Analyzed **Sales by Category**
* Analyzed **Sales by Region**
* Created **interactive slicers for filtering**

---

## 📈 Dashboard Features

The Excel dashboard includes:

* 📊 Revenue by Category Chart
* 🥇 Top 5 Products Chart
* 🌍 Sales by Region Visualization
* 🔍 Interactive **Slicers for filtering**
* 📅 Date based analysis

The dashboard allows users to dynamically explore the data.
